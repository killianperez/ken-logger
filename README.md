## LIB NEEDED
Colorama:
```
pip install colorama
```

## Note
I using numbers to improve execution speed slightly compared to strings ...

## How to use
Firstly import my file like
```python
import logger
```
Then in your code use:
```python
logger.log(<type>, <text>)
```
